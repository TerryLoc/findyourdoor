# findyourdoor.ie

Life & Mindset Coaching for Men - Terry Loughran

## Tech Stack

- React 18 + Vite
- CSS Modules
- Framer Motion
- EmailJS (contact form)
- React Router v6

## Getting Started

```bash
npm install
npm run dev
```

## Environment Variables

Copy `.env.example` to `.env` and fill in your EmailJS credentials.

## Deployment

Recommended: Netlify (drag and drop `/dist` folder or connect GitHub repo)

```bash
npm run build
```

Deploy the `/dist` folder.
